## Number Formatting Contract

Règle absolue UX-01 : toutes les valeurs numériques affichées comme intervalles `[min – max]`.

| Donnée | Format | Exemple |
|--------|--------|---------|
| Break-even (ans) | `[X – Y] ans` — 0 décimale | `[12 – 28] ans` |
| Coût total (M€) | `[X.X – Y.Y] M€` — 1 décimale | `[125.3 – 450.7] M€` |
| Spiruline (€/an) | `[X – Y] €/an` — 0 décimale | `[2 400 000 – 8 100 000] €/an` |
| Aquaculture (€/an) | `[X – Y] €/an` — 0 décimale | `[1 200 000 – 4 300 000] €/an` |
| Minéraux (€/an) | `[X – Y] €/an` — 0 décimale | `[800 000 – 3 200 000] €/an` |
| Surface agricole (km²) | `[X.X – Y.Y] km²` — 1 décimale | `[42.0 – 180.5] km²` |
| Durée de vie (ans) | `[X – Y] ans` — 0 décimale | `[35 – 80] ans` |
| Habitabilité (ans) | `[X – Y] ans` — 0 décimale | `[8 – 25] ans` |
| ROI@25/50/100 (M€) | `[X.X – Y.Y] M€` — 1 décimale | `[320.5 – 1 240.0] M€` |
| Break-even Infinity | `"—"` | `—` |

Séparateur décimal: em dash `–` (U+2013) — jamais un tiret simple `-`.

Séparateur milliers: espace fine ` ` — ou regroupement via `toLocaleString('fr-FR')` si valeur >= 1 000 000.

Source: CONTEXT.md §specifics ; pattern CalculationPanel.tsx `formatInterval`

## Interaction Contract

### Accordéon principal "Économie & ROI"

| Déclencheur | Comportement |
|-------------|-------------|
| Clic header | Toggle `isOpen` |
| Canal avec élévation sélectionné | `useEffect(() => { if (selectedCanal?.elevation) setIsOpen(true) }, [selectedCanalId])` |
| Canal désélectionné | Panel reste dans son état (pas de fermeture automatique) |
| `aria-expanded` | Mis à jour selon `isOpen` |

### Sous-accordéon "Comparer"

| Déclencheur | Comportement |
|-------------|-------------|
| Clic bouton "Comparer" | Toggle `isCompareOpen` |
| Par défaut | `isCompareOpen = false` (replié) |
| `canals.length === 1` | Bouton "Comparer" visible mais table a 1 seule ligne |
| `canals.length === 0` | Sous-accordéon masqué (pas d'accordéon sans données) |

### États de l'EconomicPanel (4 états exclusifs)

| État | Condition | Affichage |
|------|-----------|-----------|
| A — Aucun canal | `!selectedCanalId` | Texte italic `"Sélectionnez un canal pour voir l'analyse économique"` |
| B — Sans élévation | `selectedCanal && !selectedCanal.elevation` | `AlertCircle amber` + `"Chargez le profil altimétrique pour l'analyse économique"` |
| C — Sans dessalement | `elevation && !desalinationEnabled` | KPI break-even + coût construction, ROI partiel (co-produits = zéro), banner amber `"Activez les nœuds de dessalement pour les co-produits"` |
| D — Avec dessalement | `elevation && desalinationEnabled && roiResult` | Affichage complet — KPI + co-produits + ROI@25/50/100 + tableau comparatif |

Source: CONTEXT.md §decisions "Gestion de l'état dessalement désactivé"

## Accessibility Contract

| Élément | Attribut ARIA | Valeur |
|---------|--------------|--------|
| Bouton accordéon principal | `aria-expanded` | `{isOpen}` booléen |
| Bouton sous-accordéon | `aria-expanded` | `{isCompareOpen}` booléen |
| Table comparatif | `role="table"` (implicite `<table>`) | — |
| En-tête de table | `scope="col"` sur chaque `<th>` | — |
| Focus ring | `focus-visible:ring-2 focus-visible:ring-blue-400 focus-visible:ring-offset-2 focus-visible:ring-offset-gray-900` | identique aux panels existants |
| Ligne canal sélectionné | `aria-current="true"` | sur `<tr>` correspondant |

Contraste minimum: toutes les paires texte/fond respectent WCAG AA — vérifiées par audit de couleur:
- `gray-500` sur `rgba(26,26,46)` : ratio ~4.0 (AA acceptable pour texte 11px non gras)
- `white` sur `rgba(26,26,46)` : ratio ~17.0 (AAA)
- `amber-400` sur `rgba(26,26,46)` : ratio ~8.5 (AAA)

`DashboardPanel.tsx` et `EcologyPanel.tsx` utilisent `gap-[2px]` (ou `gap-0.5`) entre dt et dd dans leurs listes de définition. Pour tout nouvel élément dt/dd introduit dans `EconomicPanel`, utiliser `gap-1` (4px = xs du spacing scale). Ne pas copier mécaniquement le `gap-[2px]` des panels de référence.
