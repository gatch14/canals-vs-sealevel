### States

#### State 1 — No canal selected

```
h-10 px-4 flex items-center
  <p className="text-xs text-gray-500 italic text-center leading-relaxed w-full">
    Sélectionnez un canal pour analyser son impact écologique
  </p>
```

#### State 2 — Canal selected, no elevation profile loaded

```
h-10 px-4 flex items-center gap-1
  <AlertCircle size={12} className="text-amber-400 shrink-0" />
  <p className="text-xs text-amber-400">
    Chargez le profil altimétrique pour l'analyse écologique
  </p>
```

Uses `#F59E0B` amber — matches CalculationPanel State 2 pattern exactly.

#### State 3 — Canal + elevation loaded, no ecological impact

```
px-4 py-3
  <p className="text-xs text-gray-500 italic leading-relaxed">
    Aucune zone écologique significative sur ce tracé
  </p>
```

Condition: `ecologyResult` is non-null but `desertIntersection === null`, `endorheicAlert.detected === false`, `climateRiskFlag === false`.

#### State 4 — Desert crossing detected (ECO-01 + ECO-02)

Rendered as a `<dl>` with two `<dt>/<dd>` pairs, class pattern identical to CalculationPanel results:

```
px-4 py-1 flex flex-col gap-[2px]
  <dt className="text-[11px] text-gray-500 uppercase tracking-wider">Zone aride traversée</dt>
  <dd className="text-[13px] font-semibold text-white">
    [X – Y] km²   {/* formatInterval(result.desertIntersection.areaKm2, 'km²', 0) */}
  </dd>
```

```
px-4 py-1 flex flex-col gap-[2px]
  <dt className="text-[11px] text-gray-500 uppercase tracking-wider">Verdissement estimé</dt>
  <dd className="text-[13px] font-semibold text-white">
    [X – Y] ans   {/* formatInterval(result.greeningTimeline, 'ans', 0) */}
  </dd>
  <p className="text-[11px] text-gray-500 mt-1">
    {aridityLabel}  {/* "Hyperaride (Sahara, Atacama)" / "Aride (Sahel, steppes)" / "Semi-aride" */}
  </p>
```

Number formatting: integer display (0 decimals), `[X – Y]` with em dash U+2013 — same `formatInterval` helper as CalculationPanel.

**Aridity class labels:**
- `hyperarid` → `"Hyperaride (Sahara, Atacama, Namib)"`
- `arid` → `"Aride (Sahel, steppes d'Asie centrale)"`
- `semiarid` → `"Semi-aride"`

#### State 5 — Endorheic basin detected (ECO-03)

Rendered as a distinct alert block with red border emphasis:

```
border-t border-white/[0.06] mt-2 pt-2 px-4 pb-3 flex flex-col gap-1
  <p className="text-[11px] text-red-400 font-semibold uppercase tracking-wider flex items-center gap-1">
    <AlertCircle size={12} className="shrink-0" />
    Alerte — Bassin endorheïque
  </p>
  <p className="text-[12px] text-red-300 leading-relaxed">
    Le canal aboutit dans {basinName} — risque de salinisation irréversible
  </p>
  <p className="text-[11px] text-gray-500 leading-relaxed mt-1">
    Exemples historiques : {examples}
  </p>
```

If `basinName` is undefined: `"Le canal aboutit dans un bassin endorheïque fermé"`
If `examples` is undefined: omit the examples line.

Color: `text-red-400` (#EF4444), `text-red-300` for body text (lighter for readability).

#### State 6 — Climate risk flag (ECO-04)

Rendered as a warning block below the endorheic alert (or below desert section if no endorheic alert):

```
border-t border-white/[0.06] mt-2 pt-2 px-4 pb-3 flex flex-col gap-1
  <p className="text-[11px] text-amber-400 font-semibold uppercase tracking-wider flex items-center gap-1">
    <AlertCircle size={12} className="shrink-0" />
    Risque climatique
  </p>
  <p className="text-[12px] text-amber-300 leading-relaxed">
    Introduction d'eau dans une zone aride et chaude
  </p>
  <p className="text-[11px] text-gray-500 leading-relaxed mt-1">
    Risque de gradients de pression et de phénomènes météorologiques locaux
  </p>
```

Color: `text-amber-400` (#F59E0B), `text-amber-300` for body text.

### State Rendering Priority

States 4, 5, and 6 are **additive** — they can all render simultaneously. Rendering order:
1. Desert section (State 4) — if `desertIntersection !== null`
2. Endorheic alert (State 5) — if `endorheicAlert.detected === true`
3. Climate risk (State 6) — if `climateRiskFlag === true`

State 1, 2, 3 are **mutually exclusive** and replace the above.

## Number Formatting Contract

Reuse `formatInterval` from CalculationPanel.tsx (do not duplicate — extract to a shared helper or import):

```typescript
/** [X – Y] unité — em dash U+2013 obligatoire */
function formatInterval(iv: Interval, unit: string, decimals: number = 3): string {
  return `[${formatNumber(iv[0], decimals)} – ${formatNumber(iv[1], decimals)}] ${unit}`
}
```

For ECO-01 (area) and ECO-02 (timeline): use `decimals = 0` (integer display).

Example outputs:
- ECO-01: `[180 – 220] km²`
- ECO-02: `[20 – 50] ans`

UX-01 constraint: ECO-03 and ECO-04 are boolean flags — no numeric value displayed. Alert and flag labels are string-only. This is the documented exception to UX-01.

**Source:** CONTEXT.md D-06 "UX-01 : superficies et timelines en Interval — alerts et flags booléens autorisés", STATE.md accumulated decisions.
