## ElevationChart — Contrat Recharts

### Configuration Recharts

```tsx
<ResponsiveContainer width="100%" height={160}>
  <AreaChart
    data={points}                    // Array<{ distance: number; altitude: number }>
    margin={{ top: 8, right: 8, bottom: 0, left: 0 }}
  >
    <CartesianGrid
      strokeDasharray="3 3"
      stroke="#374151"               // gray-700
      vertical={false}               // lignes horizontales seulement
    />
    <XAxis
      dataKey="distance"
      type="number"
      domain={['dataMin', 'dataMax']}
      tickFormatter={(v) => `${v.toFixed(0)}`}
      label={{ value: 'km', position: 'insideBottomRight', offset: -4, fill: '#9CA3AF', fontSize: 10 }}
      tick={{ fill: '#9CA3AF', fontSize: 10 }}
      axisLine={{ stroke: '#374151' }}
      tickLine={{ stroke: '#374151' }}
    />
    <YAxis
      domain={['auto', 'auto']}
      tickFormatter={(v) => `${v}`}
      label={{ value: 'm', angle: -90, position: 'insideLeft', offset: 8, fill: '#9CA3AF', fontSize: 10 }}
      tick={{ fill: '#9CA3AF', fontSize: 10 }}
      axisLine={{ stroke: '#374151' }}
      tickLine={{ stroke: '#374151' }}
      width={36}
    />
    <Tooltip
      contentStyle={{
        backgroundColor: '#1F2937',
        border: '1px solid rgba(255,255,255,0.12)',
        borderRadius: '4px',
        fontSize: '11px',
        color: '#F3F4F6',
      }}
      formatter={(value: number) => [`${value.toFixed(0)} m`, 'Altitude']}
      labelFormatter={(label: number) => `Distance : ${label.toFixed(2)} km`}
    />

    {/* Zones uphill — une ReferenceArea par segment montant */}
    {uphillSegments.map((seg, i) => (
      <ReferenceArea
        key={i}
        x1={seg.distanceStart}
        x2={seg.distanceEnd}
        fill="#EF4444"
        fillOpacity={0.30}
        strokeOpacity={0}
      />
    ))}

    {/* Courbe principale */}
    <Area
      type="monotone"
      dataKey="altitude"
      stroke="#3B82F6"
      strokeWidth={2}
      fill="#3B82F6"
      fillOpacity={0.15}
      dot={false}
      activeDot={{ r: 3, fill: '#3B82F6', strokeWidth: 0 }}
      isAnimationActive={false}      // pas d'animation — outil scientifique
    />
  </AreaChart>
</ResponsiveContainer>
```

**Ordre de rendu :** Les `ReferenceArea` uphill sont déclarées AVANT la `Area` principale pour que la courbe bleue reste visible par-dessus les zones rouges.

**`isAnimationActive={false}` obligatoire :** Les animations Recharts créent un layout shift perçu et ralentissent la lecture scientifique. Désactivé sur tous les éléments.

### Type de données

```typescript
// src/types/elevation.ts
interface ElevationPoint {
  distance: number  // km depuis le début du tracé
  altitude: number  // mètres (Copernicus GLO-30)
}

interface UphillSegment {
  distanceStart: number  // km
  distanceEnd:   number  // km
  altitudeGain:  number  // m (valeur positive)
}

interface ElevationProfile {
  points:          ElevationPoint[]
  uphillSegments:  UphillSegment[]
  totalUphillGain: number   // m — somme de tous les gains
  isFullyGravity:  boolean  // true si uphillSegments.length === 0
  fetchedAt:       number   // Date.now() — pour re-fetch si reload
}
```

## Interaction Contract

### Machine d'états Phase 2 (delta sur Phase 1)

```
[État existant : SELECTION]
  → selectCanal(id)  → SELECTED + déclenche useElevation fetch automatique

[Nouvel état : SELECTED avec élévation]
  → fetch en cours    → badge "⏳ Chargement..." sur CanalListItem
                        ElevationPanel montre spinner h-40
  → fetch succès      → ElevationProfile stocké dans store
                        Badge mis à jour (✅ ou ⚠)
                        ElevationChart rendu
                        Layer canal-uphill mis à jour sur la carte
  → fetch erreur      → ElevationPanel montre état erreur
                        Pas de badge sur CanalListItem (ou badge erreur gris)
  → re-sélection du même canal → pas de re-fetch si elevation déjà chargée
  → sélection d'un autre canal → fetch si elevation absente, affiche cache si présente
```

### Déclenchement du fetch

- Le hook `useElevation` s'abonne à `selectedCanalId` dans le store Zustand.
- Quand `selectedCanalId` change vers un ID non-nul, `useElevation` vérifie `canal.elevation`.
- Si `canal.elevation` est undefined : déclenche le fetch (setElevationLoading → fetch → setElevation ou setElevationError).
- Si `canal.elevation` est défini : utilise le cache mémoire, pas de re-fetch.
- Re-fetch manuel : non disponible en Phase 2 (clic sur le canal re-sélectionne mais ne force pas).

### Layer MapLibre `canal-uphill`

- Source : `canal-uphill-source` (GeoJSON, initialisée vide au `map.load`)
- Mise à jour : via `setData()` quand `selectedCanalId` ou `canal.elevation` change dans le store
- Données : FeatureCollection de LineStrings correspondant aux segments uphill du canal sélectionné uniquement (pas tous les canaux)
- Layer ajouté après `canals-line` — visible au-dessus du bleu

**Cleanup du layer :** Quand un autre canal est sélectionné ou qu'aucun canal n'est sélectionné, `canal-uphill-source.setData({ type: 'FeatureCollection', features: [] })` pour effacer.
